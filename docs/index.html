Convolutional Neural Networks with TensorFlow for Classifying CIFAR10 Data

By Eric Scuccimarra (skooch@gmail.com)

Summary

I created and tested several convolution neural network models with TensorFlow to classify the CIFAR 10 data. The architecture and results are described below.

Training

I used the CIFAR-10 60k data, with 50,000 images used for training and the remaining 10,000 split evenly between validation and test data. The data was downloaded from http://www.cs.toronto.edu/~kriz/cifar.html.

The loss function was cross entropy, softmax was used to generate predictions and then the predictions were compared to the labels to determine accuracy.

An exponential decay was applied to the learning rate, with the starting learning rate and rate of decay varying by model.

The models were each trained for 50 epochs, using a batch size of 64. Metrics were logged to TensorBoard.

The results of the models were then compared.
